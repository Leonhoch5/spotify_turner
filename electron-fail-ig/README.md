## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/spotify_turner.git
    ```
2. Navigate to the project directory:
    ```bash
    cd spotify_turner
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

1. Start the application:
    ```bash
    npm start
    ```
